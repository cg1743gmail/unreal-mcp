========================================
触发 UE Live Coding 步骤
========================================

在 UE Editor 中：

1. 按快捷键: Ctrl + Alt + F11
   
   或
   
2. 菜单: Tools > Live Coding > Compile

========================================
等待编译完成后，MCP 命令将可用
========================================

然后运行脚本测试:
python mcp_create_bp_final.py
